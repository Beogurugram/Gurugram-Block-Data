Google Sheets Connection
Follow these steps once to connect this portal to your Google Sheet. After setup, all submissions go directly to your sheet.
1
Create a Google Sheet
Go to sheets.google.com → click + Blank → name it "School Safety QPR 2026"
→ sheets.google.com
2
Copy your Sheet ID
From the URL: docs.google.com/spreadsheets/d/YOUR_ID_HERE/edit — copy the bold part and paste below.
3
Open Apps Script
In your Google Sheet → click Extensions → Apps Script. Delete any existing code.
4
Paste the Script
Open the Code.gs file you downloaded → copy everything → paste into Apps Script editor. Replace PASTE_YOUR_SHEET_ID_HERE with your actual Sheet ID.
5
Run manualSetup()
In Apps Script, select function manualSetup from the dropdown → click ▶ Run. Approve permissions when asked. This creates the 3 sheet tabs.
6
Deploy as Web App
Click Deploy → New deployment → Web App. Set:
· Execute as: Me
· Who has access: Anyone
Click Deploy → copy the Web App URL.
7
Paste Web App URL below → Test Connection
Paste the URL you copied into the field below and click Test. A green tick means you're connected!
Apps Script Web App URL
⚠ Important: After deploying, if you edit the script, always create a New Deployment (not update) to get a fresh URL. Old URLs may cache old code.
