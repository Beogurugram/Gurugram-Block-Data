Google Sheets Connection
Paste your Apps Script Web App URL. Once connected, all config changes save to Google Sheets and load automatically for all users.

How config sync works (after connecting)

  • When you click Save All Changes → config is pushed to Google Sheets Config tab
  • When any user opens the portal → it fetches config from Google Sheets first
  • Result: change school names, deadlines, activities etc. here → everyone sees it instantly without re-uploading to GitHub
If connection fails: Go to Apps Script → Deploy → Manage deployments → confirm "Who has access" = Anyone.
